# Pet Hotel Application

This is a full-stack application for a pet hotel service with a React frontend and Express backend.

## Project Structure

- `/pet-hotel` - React frontend application
- `/backend` - Express backend API

## Setup Instructions

### Frontend

```bash
cd pet-hotel
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm run dev
```

## Technologies Used

### Frontend
- React
- React Router
- Bootstrap
- React Datepicker
- FontAwesome

### Backend
- Express
- MySQL
- Axios
- Langchain
- OpenAI API
